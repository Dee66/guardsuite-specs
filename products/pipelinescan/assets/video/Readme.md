# PipelineScan – Video Copy & Demo Assets

Source-of-truth for the PipelineScan video script, test plans, and deterministic recording information.

## Purpose
Showcases instant detection of CI/CD pipeline misconfigurations before deployment.

## Contents
- `video.yml` — Copy used for the product video.
- `demo/plan_bad.json` — Minimal CI/CD plan with 1–2 clear issues.
- `demo/plan_guard.json` — Optional advanced findings.
- `repro_notes.md` — Caret position, output hash, preset details.

## Deterministic Video Rules
- Demo plans must be version-locked.
- No changes may introduce line wrap.
- Terminal theme must use the standard preset.
- Duration target: 8–12 seconds.
